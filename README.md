# Customer Churn Analysis (Power BI)

## About this project

I created this Power BI dashboard to understand why customers are leaving (churn) and how it impacts business revenue.

This is a beginner-level project where I focused on building meaningful insights using data.

## What I did

* Calculated churn rate using DAX
* Identified high-risk customers
* Analyzed churn based on contract, internet service, and payment method
* Estimated revenue at risk

## Key Results

* Total Customers: 7032
* Churn Customers: 1869
* Churn Rate: 26.58%
* Revenue at Risk: $8M

## Insights

* Customers with month-to-month contracts are more likely to churn
* Fiber optic users have higher churn compared to others
* Electronic check payment users show higher churn
* Customers with longer tenure tend to stay

## Tools Used

* Power BI
* DAX
* CSV dataset
